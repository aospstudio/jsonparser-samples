# jsonparser-samples
 Volley JSON Parser project for your Android apps

## Requirements
```bash
Android Studio Artic Fox 2021.1.1 Canary 4 or higher,
Gradle version: 7.1.0 Alpha 04,
SDK API 31,
Platform Tools: 31.0.0,
AndroidX libraries,
Java
```
